## 📌 프로젝트 소개

**Jugland**는 간단하고 즐거운 미니게임을 한 곳에 모은 웹 기반 게임 플랫폼입니다.  
쥬니어네이버, 포켓게임, 플래시게임의 향수를 기반으로 누구나 쉽게 즐길 수 있는 게임 공간을 만드는 것이 목표입니다.

- 🎯 HTML5 / JS 기반 미니게임 허브
- 💻 React 기반의 모던 웹 UI
- 🕹️ 자체 제작 또는 오픈소스 게임 탑재
- 🔊 추억 + 감성 + 재미 = 🧠✨
