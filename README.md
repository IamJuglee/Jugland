## 📌 프로젝트 소개

**Jugland**는 간단하고 즐거운 미니게임을 한 곳에 모은 웹 기반 게임 플랫폼입니다.
누구나 쉽게 몰입하여 즐길 수 있는 게임 공간을 만드는 것이 목표입니다.

- 🎯 HTML5 / JS 기반 미니게임 허브
- 💻 React 기반의 모던 웹 UI
- 🕹️ 자체 제작 또는 오픈소스 게임 탑재
- 🔊 몰입 + 재미 + 도전 = 🧠✨

## 🚀 로컬 개발 시작하기

```bash
npm install
npm run dev
```

브라우저에서 <http://localhost:5173> 을 열어 Jugland를 확인하세요.
